THINK Back Web Services

We have made available the THINK Back toolkit as a set of web services that can be programatically accessed by your applications. These web services have been developed and integrated as part of our NCIBI Long Running Services and are available through a client side Java API.

THINK Back Java Stand-alone Implementation

A Java implementation of our techniques is available for download here. This version may be run as a stand-alone application. Notice that every individual technique can be downloaded separately, and in addition, we provide a complete suite with all the current tools and techniques that we have developed.

  • Appearance Frequency Analysis: This technique analyzes the frequency in which genes appear in biological pathways.
    • Download the latest version of our Java implementation.
    • Learn more about our technique.
  • Density Analysis: With our Density Scoring technique, the relative positions in a pathway of differentially expressed genes is taken into account. Thereby, we can identify the activation of portions of a pathway.
    • Download the latest version of our Java implementation.
    • Learn more about our technique.
